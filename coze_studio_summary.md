# Coze Studio 功能特性与部署指南

## 1. 产品概述

Coze Studio 是一站式 AI Agent 开发工具，源自服务上万家企业、数百万开发者的「扣子开发平台」，其核心引擎已完全开放。它提供可视化设计与编排工具，让 AI Agent 的创建、调试和部署变得前所未有的简单。

## 2. 技术架构

- **后端**：Golang 开发
- **前端**：React + TypeScript
- **架构模式**：微服务架构，遵循领域驱动设计 (DDD) 原则
- **核心依赖**：
  - Eino 框架：提供智能体和工作流运行时引擎
  - FlowGram：提供高质量工作流构建引擎
  - Hertz：高性能 Go HTTP 框架

## 3. 功能清单

| **功能模块** | **功能点** |
| --- | --- |
| 模型服务 | 管理模型列表，可接入 OpenAI、火山方舟等在线或离线模型服务 |
| 搭建智能体 | 编排、发布、管理智能体，支持配置工作流、知识库等资源 |
| 搭建应用 | 创建、发布应用，通过工作流搭建业务逻辑 |
| 搭建工作流 | 创建、修改、发布、删除工作流，提供可视化拖拽界面 |
| 开发资源 | 支持创建并管理插件、知识库、数据库、提示词等资源 |
| API 与 SDK | 提供创建会话、发起对话等 OpenAPI，支持通过 Chat SDK 集成到自己的应用 |

## 4. 部署方式

### 4.1 环境要求

- 最低系统要求：2 Core、4 GB 内存
- 提前安装 Docker、Docker Compose
- 启动 Docker 服务

### 4.2 部署步骤

1. **获取源码**
   ```bash
   git clone https://github.com/coze-dev/coze-studio.git
   ```

2. **配置环境变量**
   ```bash
   cd coze-studio
   # 对于 Windows 系统
   cp .env.example .env
   ```

3. **启动服务**
   ```bash
   # 对于 Windows 系统
   docker compose -f ./docker/docker-compose.yml up
   ```

4. **访问与注册**
   - 注册账号：访问 `http://localhost:8888/sign`
   - 配置模型：访问 `http://localhost:8888/admin/#model-management`
   - 访问 Coze Studio：`http://localhost:8888/`

## 5. 核心配置

### 5.1 模型配置

Coze Studio 支持多种模型接入方式：
- OpenAI
- 火山方舟 (Ark)
- Ollama
- DeepSeek
- Qwen
- Gemini

### 5.2 工作流配置

- 支持可视化拖拽设计工作流
- 支持代码节点，可执行 Python 代码
- 支持多种节点类型：条件判断、循环、并行执行等

### 5.3 知识库配置

- 支持多种文档格式导入
- 支持向量检索
- 支持 Rerank 功能
- 支持 OCR 识别

### 5.4 安全配置

- 支持用户注册开关
- 支持注册邮箱白名单
- 支持代码执行沙箱
- 支持插件授权加密
